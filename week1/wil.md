### 개발 입문 1주차 wil

#### 배포 링크
https://devintro-gamma.vercel.app/

##### 개발환경 설치

* 버전 관리/협업도구 : 깃/깃허브
* 배포: vercel
* 코드 에디터: vs code
* 페이지 구조 및 페이지 꾸미기: HTML/CSS

Git으로 프로젝트 받기
&rarr; 
pwd(print working directory): 현재 위치 출력
git clone <URL>: 깃허브 레포지토리 코드를 컨퓨터로 가져오기
ls: 지금 위치에 어떤 파일이 있는지 볼 수 있다
cd <폴더명>: 지금 위치에서 폴더 위치를 변경

##### 깃 명령어
깃은 버전 관리 프로그램, 텍스트로 명령어를 입력해서 실행
>
    git status
    git add<file>
    git commit -m"메세지"
    git push

##### HTML
인터넷에서 정보를 공유하는 통일된 텍스트 기반 양식
HTML: hyper text markup language
문서를 구조화할 수 있는 데이터가 함께 반영된 데이터
tag로 마크업할 수 있다
